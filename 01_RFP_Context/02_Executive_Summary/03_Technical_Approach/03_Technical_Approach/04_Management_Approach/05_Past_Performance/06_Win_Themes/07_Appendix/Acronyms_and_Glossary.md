## Acronyms

- AWS – Amazon Web Services
- Azure – Microsoft Azure
- CI/CD – Continuous Integration / Continuous Deployment
- DevSecOps – Development, Security, and Operations
- RBAC – Role-Based Access Control
- NPSA – NorthRiver Public Services Authority

## Glossary

**Cloud Landing Zone**  
A secure, pre-configured cloud environment that supports scalable workloads.

**DevSecOps Pipeline**  
An automated workflow that integrates development, security, and operations activities.

**Least Privilege**  
A security principle where users and systems have only the access necessary to perform their tasks.

**Migration 6R Framework**  
A model used to classify application migration strategies: Rehost, Refactor, Rearchitect, Retire, Retain, Replace.
